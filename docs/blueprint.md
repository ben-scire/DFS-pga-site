# **App Name**: PGA Contest Tracker

## Core Features:

- DraftKings API Integration: Connect to the DraftKings API to fetch live contest data for PGA tournaments.
- User Authentication: Securely authenticate and manage the 20 users participating in the contest.
- Live Leaderboard: Display a live leaderboard showing each user's current points and rank in the contest.
- Lineup Tracking: Show each user's active lineup for the current PGA tournament round.
- Data Refresh: Automatically refresh contest data at regular intervals (e.g., every minute) to provide near real-time updates.

## Style Guidelines:

- Primary color: Dark forest green (#386641) evoking the lush environment of PGA courses.
- Background color: Off-white (#F5F5DC) creating a clean and readable layout.
- Accent color: Sunset orange (#FF7F50) to highlight key information and interactive elements.
- Headline font: 'Poppins' sans-serif font for a modern and easy to read display.
- Body Font: 'PT Sans' sans-serif for clear leaderboard data
- Use golf-related icons to represent different data points and actions.
- Design a responsive layout that adapts to different screen sizes.
- Subtle transitions and animations to enhance user experience when data is refreshed.
