season-standings.json is auto-generated by the update script.

To refresh it (pull final scores from Firestore and recompute standings):

  cd league-scoring
  python3 update_league_standings.py --contest-id week-2-arnold-palmer

Replace "week-2-arnold-palmer" with the relevant contest id for the week.

Requires Firebase credentials (FIREBASE_SERVICE_ACCOUNT_JSON or GOOGLE_APPLICATION_CREDENTIALS).

Install dependencies (once):
  python3 -m pip install google-cloud-firestore google-auth
