season-standings.json is auto-generated by the update script.

To refresh it (recompute standings from weekly-scores/*.json):

  cd league-scoring
  python3 update_league_standings.py

To also pull Ben's API scores for comparison:

  python3 update_league_standings.py --ben --contest-id week-1-cognizant

Replace "week-1-cognizant" with the relevant contest id for the week.
Requires DFS API credentials in .env.local (NEXT_PUBLIC_DFS_API_BASE_URL, etc.).
